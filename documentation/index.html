Overview

  • iOS - contains template files for iOS based app.

Folder structure for assets directory

  • css - contains CSS files for the template.
  • img - contains site images.
  • js - contains JavaScript files for the template.
  • fonts - contains font icon files for the template.

Custom Classes

Use the following classes to add the required styles and effect in ease

For Android Template

.waves-effect - Adds a wave like effect to the element on click.

.waves-light - Make the waves appear more white.

.trans-effect - Adds css transition to the element. The transition is set to .3seconds.

.btm-mrgn-0 - Sets bottom margin to zero.

.white-text - Sets the text color to white.

.sec-padded-1x - Sets padding to 90px on top and bottom, 0px on left and right.

.sec-padded-2x - Sets padding to 80px on top, 0px on left, right, and bottom.

.sec-padded-3x - Sets padding to 50px on top and bottom, 0px on left and right.

.sec-padded-4x - Sets padding to 50px on top, 0px on left, right, and bottom.

.center-block - Add this class to center a block level element.

.center - Add this class to center text elements.

.overlay-shade - Adds a transparent layer of black shade.

.overlay-blue-shade - Adds a transparent layer of blue shade.

.z-depth-1 - Adds a shadow effect to the element.

.z-depth-2 - Adds a shadow effect to the element with a depth of 2.

.z-depth-3 - Adds a shadow effect to the element with a depth of 3.

.z-depth-4 - Adds a shadow effect to the element with a depth of 4.

.z-depth-5 - Adds a shadow effect to the element with a depth of 5.

.left-align - Add this class to align text elements to the left.

.right-align - Add this class to align text elements to the right.

.left - Add this class to align block level elements to left.

.right - Add this class to align block level elements to right.

For iOS

.trans-effect - Adds css transition to the element. The transition is set to .3seconds.

.btm-mrgn-0 - Sets bottom margin to zero.

.sec-padded-1x - Sets padding to 90px on top and bottom, 0px on left and right.

.sec-padded-2x - Sets padding to 80px on top, 0px on left, right, and bottom.

.sec-padded-3x - Sets padding to 50px on top and bottom, 0px on left and right.

.sec-padded-4x - Sets padding to 50px on top, 0px on left, right, and bottom.

.center - Add this class to center a block level element.

.text-center - Add this class to center text elements.

.text-left - Add this class to align text elements to the left.

.text-right - Add this class to align text elements to the right.

.colored-text - Sets text color to blue.

.bg-colored - Sets background color to blue.

.white - Sets text color to white.

.bg-white - Sets background color to white.

JavaScript

The configuration to all javascripts is stored in js/config.js
Always add new javascript files before the closing </body> tag.
Use jQuery scrict mode.

$(document).ready(function() {
    'use strict';
});

Smooth Scroll - Change speed to increase or decrease the click to scroll speed.

$('.btn-link,.navbar .nav a').smoothScroll({offset: -50, speed: 1200});


Typing effect - Add or remove words to the string array to make changes. You can also set the loop to "true" or "false" to change repetitions.

$(".type").typed({
    strings: ["Sweet", "Modern", "Beautiful", "Elegant", "Cool", "Extraordinary", "Impressive"],
    typeSpeed: 200,
    backDelay: 1500,
    loop: true,
    loopCount: false
});


Google Map

Add your location to the 'center' and 'position' variable in terms of latitude and longitude.

function initMap() {
    var mapOptions = {
    zoom: 13,
    scrollwheel: false,
    center: new google.maps.LatLng(40.693629, -73.938336),
}

var marker = new google.maps.Marker({
position: new google.maps.LatLng(40.693629, -73.938336),
}
